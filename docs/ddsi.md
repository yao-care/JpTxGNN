# 薬物-疾患注意事項（DDSI）

<p class="key-answer" data-question="DDSI とは？">
DDSI（Drug-Disease Interaction）は、特定の疾患を持つ患者に対して、特定の医薬品の使用が禁忌または注意が必要な場合の情報です。
</p>

---

## 概要

ドラッグリポジショニング候補が患者の既存疾患に対して禁忌でないかを確認することは重要です。

## 禁忌の種類

| 種類 | 説明 |
|------|------|
| **絶対禁忌** | いかなる状況でも使用不可 |
| **相対禁忌** | 特定の条件下で使用注意 |
| **慎重投与** | 注意深いモニタリングが必要 |
