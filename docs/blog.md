## プロジェクト概要

JpTxGNN は、日本 PMDA 承認医薬品を対象としたドラッグリポジショニング予測システムです。

### 主な特徴

- **TxGNN 知識グラフ**: 17,000+ 疾患、8,000+ 薬物の関係データ
- **深層学習予測**: 新規薬物-疾患関連の予測
- **FHIR 準拠**: HL7 FHIR R4 標準の API 提供
- **エビデンス収集**: PubMed、ClinicalTrials.gov からの自動収集
