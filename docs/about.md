# JpTxGNN について

## 背景

医薬品リポジショニング（ドラッグリポジショニング）は、既存の承認医薬品に新たな適応症を見出す手法です。新薬開発に比べて開発期間とコストを大幅に削減できる可能性があります。

## 技術スタック

### TxGNN 知識グラフ

TxGNN (Therapeutic Knowledge Graph Neural Network) は、医薬品と疾病の関係を予測するための知識グラフベースの機械学習モデルです。

- **ノード数**: 約129,000
- **エッジ数**: 約8,000,000
- **関係タイプ**: indication, contraindication, off-label use など

### データソース

| ソース | 説明 |
|--------|------|
| SSK 医薬品マスター | 薬価基準コード、薬価、単位情報 |
| KEGG DRUG | 効能・効果、分子式、ATC コード |
| DrugBank | 国際標準医薬品識別子 |
| TxGNN | 薬物-疾病関係の知識グラフ |

## 免責事項

本プロジェクトの結果は研究参考のみを目的としており、医療アドバイスを構成するものではありません。

- 予測結果は臨床検証を経ていません
- 医療判断には使用しないでください
- 専門家の助言を必ず求めてください
